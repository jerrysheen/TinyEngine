1. **硬编码的灵活性损失**: 固定的 space/slot 分配限制了未来扩展
2. **向后兼容性**: 之前使用泛化配置的代码可能失效

2. **遗留泛化代码**: 未完全清理的泛化配置代码导致不一致
3. **Shader 与 Runtime 不匹配**: Shader 中的 space/slot 声明与 Runtime 硬编码不一致
