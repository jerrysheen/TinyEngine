## 风险点
1. **DrawIndirect 数据准备**: Indirect 参数列表的数据填充正确性
2. **PerFrameAllocator 限制**: 每帧分配的 buffer 大小限制
3. **Instance 数据对齐**: GPU 对 instance 数据的严格对齐要求
4. **Batch 合并边界**: 哪些 drawcall 可以合并，哪些必须分开

## 可能的错误原因
1. **Indirect Args 格式错误**: DrawIndirectList 的参数格式与 D3D12 要求不匹配
2. **Instance Count 溢出**: 单次绘制的 instance 数量超过限制
3. **Buffer 地址计算错误**: LinearAllocateBuffer 的地址计算有误
4. **RenderContext 状态错乱**: Batch 渲染流程中的状态切换问题
5. **Resource Barrier 缺失**: Indirect Buffer 的 GPU 可见性未正确设置
