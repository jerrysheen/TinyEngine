# 已知技术债（待处理）

1. D3D12 绑定开销
说明：`Runtime/Platforms/D3D12/D3D12RenderAPI.cpp` 的 `RenderAPISetMaterial` 每次都会重复绑定全局 SRV（AllObjectData / AllMaterialData / Visibility），即使 RootSignature 未切换也会重复执行。
影响：CPU 端绑定与 descriptor 复制开销偏大。
建议：将全局 SRV 绑定移到 `RenderAPISetRenderState`（RootSig 变更时）或 `RenderAPIBeginFrame`，并在 `RenderAPISetMaterial` 做材质缓存，减少重复绑定。
